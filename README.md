# Assembly Language Programs (8086)

This repository contains basic programs written in **8086 Assembly Language** for learning and practice.

## 📚 Topics Covered

* Data transfer instructions
* Arithmetic operations
* Logical operations
* Loops and control flow
* String operations

## 🛠️ Requirements

* Emulator: EMU8086 / DOSBox / MASM / TASM
* Basic understanding of assembly syntax



## 📁 Structure

Each file contains a separate program demonstrating a specific concept.

## 🎯 Purpose

To build a strong foundation in low-level programming and understand how programs interact with hardware.
